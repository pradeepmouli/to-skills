# Functions

## extensions

### `normalizeExtensionPriority`

Get the priority for an extension.

```ts
normalizeExtensionPriority(ext: any, defaultPriority: number): number
```

**Parameters:**

- `ext: any` — Any extension
- `defaultPriority: number` — Fallback priority if none is defined.
  **Returns:** `number` — The priority for the extension.
