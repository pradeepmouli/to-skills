# @to-skills/mcp

> Extract and bundle MCP servers as Agent Skills — one extractor, every agent.

Part of the [to-skills](https://github.com/pradeepmouli/to-skills) ecosystem. Introspects live Model Context Protocol (MCP) servers via stdio or HTTP and produces progressive-disclosure `SKILL.md` output consumable by MCP-native agents (via `mcp:` frontmatter) or non-MCP agents (via CLI-as-proxy adapters).

Generated from `specs/001-mcp-extract-bundle`. See the spec directory for design docs, research, contracts, and the full task list.

Scaffolding only. Implementation tracked in `specs/001-mcp-extract-bundle/tasks.md`.
