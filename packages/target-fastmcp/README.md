# @to-skills/target-fastmcp

> CLI-as-proxy invocation-target adapter for `@to-skills/mcp`. Emits shell-command skills that use the FastMCP Python CLI (`fastmcp list`, `fastmcp call`) to invoke MCP tools.

Part of the [to-skills](https://github.com/pradeepmouli/to-skills) ecosystem. The CLI terminates the MCP protocol at the shell boundary; the agent never sees MCP.

Target CLI: `fastmcp` (Python, installed via `pip install fastmcp`).

Generated from `specs/001-mcp-extract-bundle`.
