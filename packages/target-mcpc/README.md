# @to-skills/target-mcpc

> CLI-as-proxy invocation-target adapter for `@to-skills/mcp`. Emits shell-command skills that use Apify's [`mcpc`](https://github.com/apify/mcpc) CLI to invoke MCP tools — consumable by any agent with a shell tool.

Part of the [to-skills](https://github.com/pradeepmouli/to-skills) ecosystem. The CLI terminates the MCP protocol at the shell boundary; the agent never sees MCP.

Target CLI: `mcpc@^2.1`.

Generated from `specs/001-mcp-extract-bundle`.
